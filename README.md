# 🧟 Final Destination

> 3D Zombie Shooter Game

Final Destination은 좀비 무리를 돌파하며 탈출 지점까지 도달하는 것을 목표로 하는 **3D 좀비 슈팅 게임**입니다.  
플레이어는 다양한 무기와 아이템을 활용하여 몰려오는 좀비들을 처치하며 맵을 진행해야 합니다.

# 🎮 Game Overview

플레이어는 맵을 탐험하며 끊임없이 등장하는 좀비들과 전투를 벌이고  
맵 곳곳에 배치된 **무기와 아이템을 활용하여 생존하며 탈출 지점까지 도달해야 합니다.**

게임 진행 중에는 다양한 특성을 가진 좀비들이 등장하며  
각 좀비의 특성에 맞게 전략적으로 대응해야 합니다.

# 🧟 Zombie System

게임에는 서로 다른 특성을 가진 좀비들이 등장합니다.

| 좀비 타입 | 특징 |
|----------|------|
| 일반형 좀비 | 기본적인 행동 패턴 |
| 속도형 좀비 | 빠른 이동 속도 |
| 공격형 좀비 | 높은 공격력 |

---

# 🗺 Game Objective

맵을 진행하며 등장하는 좀비들을 처치하고  
다양한 무기와 아이템을 활용하여 **최종 탈출 지점까지 도달하는 것이 목표입니다.**
